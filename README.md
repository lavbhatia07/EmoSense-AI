# EmoSense AI 🎭🤖

An AI-powered Emotion Detection web application built using Machine Learning, NLP, and Flask. The app analyzes user text and predicts emotions such as Joy, Sadness, Anger, Fear, Love, and Surprise in real time through a modern and interactive web interface.

---

## 🚀 Live Demo

https://emosense-ai-6jmm.onrender.com/

---

## ✨ Features

- Real-time emotion prediction
- Predicts 6 emotions:
  - 😊 Joy
  - 😢 Sadness
  - 😠 Anger
  - ❤️ Love
  - 😨 Fear
- NLP-based text preprocessing
- Machine Learning prediction pipeline
- Modern responsive UI
- Emotion-specific colors and emojis
- Confidence score display
- Flask backend integration
- AJAX-based predictions without page reload
- Deployment-ready project structure
